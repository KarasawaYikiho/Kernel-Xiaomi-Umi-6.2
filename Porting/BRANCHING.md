# Branching Strategy

- `main` — Integration branch
- `port/phase0-*` — Baseline lock
- `port/phase1-*` — Inventory/gap analysis
- `port/phase2-*` — Migration/build
- `port/phase3-*` — Feature completion
- `port/hotfix-*` — Urgent fixes
