# Security Policy

## Scope

Kernel source, DTS, defconfig, drivers, GitHub workflows, scripts, and documentation.

## Reporting

Open a private security advisory on GitHub. Include:
- Affected file(s)
- Impact description
- Reproduction steps
- Suggested fix (optional)

## Response

- Triage: within 72 hours
- Patched issues noted in `Porting/CHANGELOG.md`

## Rules

- Never commit tokens/credentials
- Keep artifacts free of secrets
